# Pako & Bella

Projeto de identidade visual para pet shop fictício inspirado em uma estética acolhedora, afetiva e moderna.

## Conceito
A marca busca transmitir carinho, cuidado e conexão entre pets e seus donos através de uma identidade visual suave e amigável.

## Ferramentas utilizadas
- Illustrator
- Photoshop
- Canva
